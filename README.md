🤖 JIAT AI Code Assistant

An AI-powered code assistant that analyzes your code, detects errors, suggests fixes, and shows output — all in one place.

---

🚀 Features

- 🔍 Detect syntax and logical errors
- 📍 Show error location and line number
- 🛠 Suggest fixes with explanations
- ✅ Display output if code is correct
- 💡 Provide improvements
- 🧠 AI-powered using OpenRouter API
- 🧾 Code editor with line numbers (CodeMirror)
- 🎨 Clean UI with TailwindCSS

---

🧪 How It Works

1. Paste or write your code
2. Select programming language
3. Click **Analyze**
4. Get:
   - Errors (if any)
   - Fix suggestions
   - Output (if correct)

---

🛠 Technologies Used

- HTML
- CSS (TailwindCSS)
- JavaScript
- CodeMirror Editor
- OpenRouter API
